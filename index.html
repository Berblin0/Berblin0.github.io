linux解压和压缩命令
更新于 2019-10-06
linux
linux学习

1、.tar
解包：tar xvf FileName.tar
打包：tar cvf FileName.tar DirName
（注：tar是打包，不是压缩！）
———————————————

2、.zip
解压：unzip FileName.zip
压缩：zip FileName.zip DirName
———————————————
3、.rar
解压：rar x FileName.rar
压缩：rar a FileName.rar DirName

———————————————

4、.gz
解压1：gunzip FileName.gz
解压2：gzip -d FileName.gz
压缩：gzip FileName

———————————————

5、.tar.gz 和 .tgz
解压：tar zxvf FileName.tar.gz
压缩：tar zcvf FileName.tar.gz DirName
———————————————
6、.bz2
解压1：bzip2 -d FileName.bz2
解压2：bunzip2 FileName.bz2
压缩： bzip2 -z FileName

———————————————

7、.tar.bz2
解压：tar jxvf FileName.tar.bz2
压缩：tar jcvf FileName.tar.bz2 DirName
———————————————
8、.bz
解压1：bzip2 -d FileName.bz
解压2：bunzip2 FileName.bz
压缩：未知

———————————————

9、.tar.bz
解压：tar jxvf FileName.tar.bz
压缩：未知
———————————————
10、.Z
解压：uncompress FileName.Z
压缩：compress FileName

———————————————

11、.tar.Z

解压：tar Zxvf FileName.tar.Z
压缩：tar Zcvf FileName.tar.Z DirName
———————————————

12、.lha
解压：lha -e FileName.lha
压缩：lha -a FileName.lha FileName
———————————————

13.rpm
解包：rpm2cpio FileName.rpm | cpio -div
———————————————
14、.deb
解包：ar p FileName.deb data.tar.gz | tar zxf -

摘录

幽静的树林里分出两条路，

可惜我不能同时去涉足，

我在那路口久久伫立，

我向着一条路极目望去，

直到它消失在丛林深处。

但我却选了另外一条路，

它荒草萋萋，十分幽寂；

显得更诱人、更美丽，

虽然在这两条小路上，

都很少留下旅人的足迹。

虽然那天清晨落叶满地，

两条路都未见脚印痕迹。

呵，留下一条路等改日再见！

但我知道路径延绵无尽头，

恐怕我难以再回返。

也许多少年后在某个地方，

我将轻声叹息把往事回顾：

一片树林里分出两条路，

而我选了人迹更少的一条，

从此决定了我一生的道路。

人生如逆旅，我亦是行人。

朋友们，江湖再见！

​ –摘自一位大神

Python编程

1、变量的使用

2、使用制表符或换行符来添加空白

要在字符串中添加制表符，可使用字符组合 \t ；

要在字符串中添加换行符，可使用字符组合 \n ;

字符串中同时使用制表符和换行符组合 \n\t 让字符换到下一行开头添加一个制表符。

删除空白

删除字符串中开头的空白，可使用方法 lstrip()

删除字符串中末尾的空白，可使用方法 rstrip()

同时删除字符串两端的空白，可使用方法 strip()

3、添加和删除元素

append() 将元素添加到列表尾部，而不会影响列表中的其他元素

insert() 将元素插入列表中，需要指定新元素的索引和值

del 删除列表中的元素，需要确定元素在列表中的位置

pop() 删除（弹出）列表末尾的元素，pop(0) 也可以指定删（除弹）出列表中任何位置的元素，需要在括号里指定要（删除）的元素的索引

remove() 从列表中删除已知元素的值

4、列表排序

sort() 让列表中元素按字母顺序排列

sort(reverse=True) 按字母顺序相反的顺序排列元素

sorted() 对列表进行临时排序

sorted(reverse=True)

reverse() 反转列表元素的排列顺序，方法reverse()永久性地修改列表元素的排列顺序，但可以恢复到原来的排列顺序，只需对列表再次调用reverse()

5、创建数值列表

range(1,5) 打印1~4的数值

list(range(1,5)) 打印列表

（**）表示乘方运算

6、if elif else语句

if

if-else

if-elif-else

多个elif

7、字典

方法 items() 返回一个键-值对列表

不需要使用字典中的值时，可用方法 keys()

for name in favorite_languages.keys(): 也可以替换为 for name in favorite_languages:输出结果相同。

values() 只返回字典中的值列表

set() 剔除重复项

8、用户输入和while循环

函数input() 在Python2.7中，会将用户输入解读为Python代码，并运行。需要使用函数 raw_input()来获取输入

在将数值输入用于计算或比较前，务必要将其转换为数值表示。

continue 语句跳出本次循环 break跳出整个循环

9、base64

base64编码

进行编码时，要对字符串进行utf-8编码，否则会出现报错。

base64解码

str.format() 基本语法是通过 { }和 :

format 函数可以接受不限个参数，位置可以不按顺序

也可以设置参数

11、random随机数模块

random() 生成随机小数，默认0~1之间

randint() 生成随机整数

choie() 随机抽取列表中的一个元素

12、文件读取

进行文件读取有两种方式

1、使用open() 函数

2、使用with open 方式

“+”表示可同时读写某个文件

1、r+ 可读写文件

2、w+ 写读

3、a+ 追加

open() 函数用于打开一个文件，并返回文件对象，如果文件无法打开，则会OSError异常报错，使用open() 函数需要手动关闭文件。

语法：open(file,mode) mode表示为要以什么模式打开

读取文件内容可指定每次读取字符

readline() 每次读取一行内容(比较适合读取大文件)

readlines() 读取文件所有内容，并保存在列表(list)变量中，每行作为一个元素。

seek() 方法用于移动文件读取指针到指定位置

linecache.getline(file,n) n表示要读取的第几行内容

使用with open 方式，则不需要手动关闭文件

文件写入

write() 函数将字符串写入文件中(默认是不换行写入的)

w+ 模式的话，会将文件之前的内容覆盖掉，重新写入。

文件删除

文件删除需要使用os模块的remove()函数
